View signature

As a customer
I want to see reasons my uploaded signature was rejected
So that I can provide a better signature next time

Scenario: AC-01 - Return to dashboard when app cancelled

And
I submit the following task outcome:
Task outcome Rejection reason Task subject type New task type New task status
Rejected Signature not clear Signature Replace image Active
And
the application is cancelled
And
I have accessed the applicant portal on GOV.UK
And
I have accessed the "Your provisional licence application" page
And
I click the "View Signature" link
