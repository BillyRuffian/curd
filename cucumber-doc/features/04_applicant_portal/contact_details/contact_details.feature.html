Contact Details

As a customer with a submitted application for a first provisional
I want to see the contact information currently held for me
So that I know DVLA will be able to communicate with me

Scenario: AC-01,02,03 Display 'Update contact information' page with existing data

When
I have accessed the "Update contact information" page
Then
the existing contact details are displayed

Scenario: AC-04 Display text 'Not provided' when no mobile number exists on the application

And
I have accessed the "Contact details" page
And
I have left the mobile number field blank and continue
When
I have accessed the "Update contact information" page
Then
the mobile number displayed is set to 'Not Provided'

Scenario: AC-05 Change Link

And
I have accessed the "Update contact information" page
When
I click on Change
Then
I am redirected to the "Contact details" page

Scenario: AC-06 Return to dashboard

And
I have accessed the "Update contact information" page
When
I click the "Return to dashboard" link
Then
I am redirected to the "Your provisional licence application" page

Scenario: AC-07 Display 'Contact details' page with existing data

When
I have accessed the "Contact details" page
Then
the fields are pre-populated with the existing data

Scenario: AC-08,13 Mobile number is blank when no mobile number exists on the application

And
I have accessed the "Contact details" page
And
I have left the mobile number field blank and continue

Scenario: AC-09c Invalid max length of the Email address

And
I have accessed the "Contact details" page
And
I enter an invalid max length email address
When
I proceed
Then
the following error is present:
Attribute Error
Error Summary Email address cannot be longer than 254 characters
Error Message Email address cannot be longer than 254 characters
And
clicking on the error link will focus the cursor into the corresponding field:
Error text Field ID
Email address cannot be longer than 254 characters email_address
