Application statuses

As a customer who has submitted a First Provisional Application
I want to be able to see the status of my app
So that I know when to expect my provisional

AC-02 cannot be automated - decided to be a manual test

Scenario: AC-01a - Page content - Static text

Assert that static text on screen matches screen design

Given
an application exists with user uploaded images
And
I submit the following task outcome:
Task outcome Rejection reason Task subject type New task type New task status
Rejected Signature not clear Signature Replace image Active
And
I have accessed the applicant portal on GOV.UK
