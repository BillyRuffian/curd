Allow Minor User Input Variations In Identity Verification

As a customer
I want to be allowed to sign into the applicant portal
Despite making small casing and spacing mistakes when entering my identity information
So that I can access the portal to view the status of my application

Scenario Outline: AC-04a - Sign in with different character casing in names

As a customer
If I attempt to sign into the applicant portal with the casing of one of my names inverted
I expect to be able to sign into the portal as normal

Given
an application exists
And
I have accessed the applicant portal on GOV.UK
And
I have accessed the "Your Details" page
When
I try to sign in using identity details that have <casing difference>
Then
I am presented with the "Your provisional licence application" page with the path "/applicant_portal/dashboard"
Examples
casing difference
inverted first name casing
inverted last name casing

Scenario: AC-04b - Sign in with different first-name character casing if I have two first names

As a customer with two first names
If I attempt to sign into the applicant portal with the casing of both of my first names inverted
I expect to be able to sign into the portal as normal

Given
an application exists for a person who has two first names
And
I have accessed the applicant portal on GOV.UK
And
I have accessed the "Your Details" page
When
I try to sign in using identity details that have inverted first names casing
Then
I am presented with the "Your provisional licence application" page with the path "/applicant_portal/dashboard"

Scenario: AC-04c - Sign in with the initial of the second of my two first names

As a customer with two first names
If I attempt to sign into the applicant portal using the initial of the second of my two first names
I expect to be able to sign into the portal as normal

Given
an application exists for a person who has two first names
And
I have accessed the applicant portal on GOV.UK
And

When
I try to sign in using identity details that have my full first name but only the initial of my second first name
