Sign In

As a customer who has completed a provisional licence application
I want to sign into the Applicant Portal
So that I can check the progress of my application

Scenario: 01 Sign in and verify and navigate back to sign in using browser back button

And
I have accessed the "Your provisional licence application" page
When
I navigate backwards in the browser 10 times
Then
I am redirected to the "Your provisional licence application" page

Scenario: 02 Sign in and verify and navigate directly to sign in

And
I have accessed the "Your provisional licence application" page
When
I have accessed the applicant portal on GOV.UK
Then
I am redirected to the "Your provisional licence application" page

Scenario Outline: AC-02 Sanitise the application reference number

And
I have accessed the "Sign in" page
And
I enter an application reference number with <reference number> and continue
And
I am redirected to the "Your details" page
When
I click the "Back" link
Then
the sanitised application reference number is displayed
Examples
reference number
leading whitespace
trailing whitespace
lowercase characters

Scenario: AC-03a Application reference number validation - blank

And
I have accessed the "Sign in" page
When
I submit a blank application reference number
Then
the following error is present:
Attribute Error
Error Summary Enter your application reference number
Error Message Enter your application reference number
And
clicking on the error link will focus the cursor into the corresponding field:
Error text Field ID
Enter your application reference number customer_ref_num

Scenario: AC-03b Application reference number validation

When
I have accessed the "Sign in" page
Then
entering an invalid application reference number and continuing will invoke the following errors
Invalid data Attribute Error Focusable Field
Length Error Summary Your reference number must be 14 characters long, including dashes customer_ref_num
Length and characters Error Summary Your reference number must be 14 characters long, including dashes customer_ref_num
Length, characters and format Error Summary Your reference number must be 14 characters long, including dashes customer_ref_num
Characters Error Summary Reference number must only include letters and dashes customer_ref_num
Characters and format Error Summary Reference number must only include letters and dashes customer_ref_num
Format Error Summary Enter a valid reference number customer_ref_num

Scenario: AC-04 Continue to the 'Your details' page if not yet verified.

And
I have accessed the "Sign in" page
When
I enter a valid application reference number and continue
Then
I am redirected to the "Your details" page

Scenario: AC-05 Already identified - Continue to the dashboard after signed in with valid application reference number.

And
I have accessed the "Sign in" page
And
I sign in using an application reference number that does not match my identity and continue
And
I have accessed the "Unable to find your application" page
And
I click on Try again
When
I enter a valid application reference number and continue
Then
I am redirected to the "Your provisional licence application" page
