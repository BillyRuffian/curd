Do you hold a GB Driving Licence

As a user wanting to confirm a provisional applicant’s identity
I want to confirm whether or not I hold a full valid GB photocard driving licence
So that I can determine whether or not I am eligible to confirm the applicant's identity

Scenario: AC-02 User selects “Yes”

And
the following page content is not present:
Attribute Value
Input ID Driving licence number
When
I select 'Yes'
Then
the following page content is present:
Attribute Value
Input ID Driving licence number

Scenario: AC-03 User first selects “Yes” and then selects “No”

And
I select 'Yes'
When
I select 'No'
Then
the following page content is not present:
Attribute Value
Input ID Driving licence number

Scenario: AC-04 User enters information, selects “No” and then selects "Yes"

And
I enter a valid uppercase driving licence
And
I select 'No'
When
I select 'Yes'
Then
the "Driving licence number" field is empty

Scenario: AC-06 User selects “Yes” and does not enter a driving licence number

When
I select 'Yes' and proceed
Then
the following page content is present:
Attribute Value
Error Summary Enter your driving licence number
Error Message Enter your driving licence number
And
clicking on the error link will focus the cursor into the corresponding field:
Error text Field ID
Enter your driving licence number Driving Licence Number

Scenario: AC-07 User selects “Yes” and enters a driving licence number

When
I enter a valid uppercase driving licence and continue
Then
I am presented with the "Do you work in or are you retired from a suitable profession?" page

Scenario: AC-08 and AC-09 User selects “No” and then selects “Continue” and then selects "Return to GOV.UK"

And
I select 'No' and proceed
And
I am presented with the "Thank you" page
When
I click on Return to GOV.UK
Then
I am presented with the page with url "https://www.gov.uk/"
