What is your postcode

As a user wanting to confirm a provisional applicant’s identity
I want to find my address by the postcode and house number
So that I can provide my address and continue with the confirmation without manually entering my address details

Scenario: AC-02a Invalid Content in "Postcode" field - Invalid character

An invalid postcode is one that is not alphanumeric
Enter only 1 invalid character

When
I enter a non-alphanumeric character into the postcode field
Then
that invalid character does not appear in the postcode field
And
no error will be displayed

Scenario: AC-02b Invalid Content in "Postcode" field - Invalid postcode

An invalid postcode is one that is not alphanumeric.
Enter a full postcode with one or more invalid chars

When
I enter an invalid postcode into the postcode field
Then
the invalid characters do not appear in the postcode field
And
no error will be displayed

Scenario: AC-03 Lowercase characters in "Postcode" field

Entering a lowercase postcode will appear as uppercase in
the postcode field

When
I enter a postcode in lowercase
Then
the postcode appears as uppercase in the input field
And
no error will be displayed

Scenario: AC-05 User enters a valid postcode and selects continue

Optional house number will not be entered in this scenario

When
I enter a valid postcode and click on "Continue"
Then
I am redirected to the "Your contact details" page

Scenario: AC-06 User has not entered a postcode and selects "Find UK address" button

Submitting a blank form will produce a validation error

When
I enter a blank postcode form and click on "Continue"
Then
the following page content is present:
Attribute Value
Error Summary Postcode must be between five and eight characters and in a valid format, e.g. AB1 2BA or AB12BA

Scenario: AC-6a Click on error link will focus cursor to field

An error message appears and the user clicks on the error in the summary
resulting in the cursor moving to the corresponding field

When
I enter a blank postcode form and click on "Continue"
Then
clicking on the error link will focus the cursor into the corresponding field:
Error text Field ID
Postcode must be between five and eight characters and in a valid format, e.g. AB1 2BA or AB12BA postcode

Scenario: AC-07 User enters a valid postcode and house number and selects continue with one exact match

Note: A single match postcode is one that only returns a single address

When
I enter a single match postcode and click on "Continue"
Then
I am presented with the "Your contact details" page
