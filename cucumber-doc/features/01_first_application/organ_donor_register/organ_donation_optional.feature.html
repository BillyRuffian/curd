Organ Donation Optional

As a customer wishing to apply for a provisional licence online
I want to select which organs I want to donate opting in to the NHS Organ Donor Register
So that I am able to only donate the organs I wish to

Scenario: AC-01 Display "Organ Donation (optional)" Content

When
I have accessed the "Organ Donation Optional" page
Then
the correct content is present for "Organ Donation (optional)"

Scenario: AC-02 User selects an option and then selects "continue" while they have a valid passport photo

And
I have accessed the "Organ Donation Optional" page
When
I check the "All_organs" checkbox and continue
Then
I am presented with the "You will now need to provide a photo and signature" page

Scenario: AC-04 User has not selected an option

And
I have accessed the "Organ Donation Optional" page
When
I do not select an option and then select continue
Then
the following error is present:
Attribute Error
Error Summary Select one or more options below
Error Message Select one or more options below

Scenario: AC-05 User selects multiple individual options

And
I have accessed the "Organ Donation Optional" page
When
I select random individual organs
Then
those organs will be checked

Scenario: AC-06 User selects "Any of my organs and tissues" and then another option

And
I have accessed the "Organ Donation Optional" page
And
I have checked the "All_organs" checkbox
When
I check the "Corneas" checkbox
Then
the "All_organs" checkbox will be unchecked

Scenario: AC-07 User selects any options and then "Any of my organs and tissues"

And
I have accessed the "Organ Donation Optional" page
And
I have checked the "Corneas" checkbox
When
I check the "All_organs" checkbox
Then
the "Corneas" checkbox will be unchecked
