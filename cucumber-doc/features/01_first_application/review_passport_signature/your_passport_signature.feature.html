Your Passport Signature

As a customer wishing to apply for a provisional licence online
I want to be able to use my passport signature on my provisional licence if I choose to
So that I do not need to spend time providing a new signature

Scenario: AC-01 Display "Do you want to use this signature from your passport on your driving licence?" Content

When
the correct content is present for "Do you want to use this signature from your passport on your driving licence?"
Then
the user's passport signature will be displayed as a preview

Scenario: AC-02 User selects yes and then selects continue

When
I select 'Yes' and proceed
Then
I am presented with the "Check your answers before sending your application" page

Scenario: AC-03 User selects no and then selects continue

When
I select 'No, I'll provide my own signature' and proceed
Then
I am presented with the "Rules for driving licence signatures" page

Scenario: AC-04 User has not selected an option

When
I proceed
Then
the following error is present:
Attribute Error
Error Summary Select yes if you want to use this signature on your driving licence
Error Message Select yes if you want to use this signature on your driving licence
And
clicking on the error link will focus the cursor into the corresponding field:
Error text Field ID Visible
Select yes if you want to use this signature on your driving licence passport_signature_decision True
