Which country were you born in

As a customer wishing to apply for a provisional licence online
I want to provide which country I was born in, during my application
So that I can provide the correct information for my licence application

Scenario: AC-01 - Display "What country were you born in?" Content

When
I have accessed the "Which country were you born in?" page
Then
the correct content is present for "Which country were you born in?"

Scenario: AC-02.1 User begins to enter country

When
I begin typing in the non UK country text box
Then
a drop-down list of non UK countries which match their current text entry will appear below

Scenario: AC-02.2 User begins to enter country

And
I begin typing in the country text box
And
a drop-down list of countries which match their current text entry will appear below
When
I continue typing the country
Then
the suggestions will dynamically be updated to continue to provide a best match

Scenario: AC-03 - User selects country from drop-down list

And
I begin typing in the country text box
And
a drop-down list of countries which match their current text entry will appear below
When
I select an option from that drop-down list
Then
the text box will be populated by my selection

Scenario: AC-04 - User has entered a country and selects "Continue"

And
I begin typing in the country text box
And
I select an option from that drop-down list
When
I proceed
Then
I am redirected to the "Enter your UK passport number" page

Scenario: AC05 - User has not made an entry and selects "Continue"

When
I proceed
Then
the following error is present:
