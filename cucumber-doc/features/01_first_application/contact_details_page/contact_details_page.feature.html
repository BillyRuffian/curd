Contact Details Page

As a customer wishing to apply for a provisional licence online
I want to enter my contact details when applying for a licence online
So that I can be kept up to date on my application and be contacted if there is an issue

Scenario: AC-01, AC-02 "Contact Details" Content

When
I have accessed the "Contact details" page
Then
the correct content is present for "Contact details"

Scenario: AC-04 Display page content - No return dropout: Browser back button

And
I have accessed the "Contact details" page
When
I navigate backwards in the browser
Then
the correct content is present for "You're unable to go to the previous page"

Scenario: AC-02 User clicks collapsed "how we use your contact details" link

And
I have accessed the "Contact details" page
And
the details are hidden
When
I expand the details
Then
the expanded text is displayed

Scenario: AC-03 User clicks expanded "how we use your contact details" link

When
I have accessed the "Contact details" page
Then
the details are hidden

Scenario: AC-04 User has not entered an email address and selects continue

When
I have accessed the "Contact details" page
Then
leaving a field blank and continuing will invoke the following errors:
Blank fields Attribute Error Focusable field
Email address, Mobile number Error Summary Enter an email address Email address
Email address, Mobile number Error Message Enter an email address
Email address Error Summary Enter an email address Email address

Scenario: AC-05 User enters an email address and selects continue

And
I have accessed the "Contact details" page
And
I have entered my email address
When
I proceed
Then
I am redirected to the "Would you like to be on the NHS Organ Donor Register?" page

Scenario Outline: AC-04, AC-06 User enters a valid email address and selects "Continue"

And
I have accessed the "Contact details" page
And
I entered valid allowable <validation_rules> email address in the email address field
When
I proceed
Then
I am redirected to the "Would you like to be on the NHS Organ Donor Register?" page
Examples
validation_rules
special character
min character length
max character length
email with extra spaces
